# drafts/

Freeform scratch space for a run: the candidate-pool dump, working Lark DocxXML, skeleton and section fragments — anything intermediate.

Overwrite freely; nothing here is durable or tracked. The final delivered DocxXML is archived to `../reports/` (`YYYY-MM-DD-<slug>.docxxml`), and the readable analysis lives in `runs/`. Contents are gitignored; this README is tracked to keep the directory present on clone.
